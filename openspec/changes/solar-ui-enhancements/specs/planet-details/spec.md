## MODIFIED Requirements

### Requirement: Panel Content

The panel SHALL display dynamic content based on the selected planet, featuring both English and Traditional Chinese text.

#### Scenario: Display planet data in dual languages

- **WHEN** user clicks on a planet (e.g., Mars)
- **THEN** the panel MUST show "Mars / 火星" as the title and display scientific data (Diameter, Distance, etc.) with labels in both English and Traditional Chinese.
